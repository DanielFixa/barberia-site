# Barberia Site

Um site moderno para barbearia desenvolvido com Next.js, React e Tailwind CSS.

## Descrição

Este é um site profissional para barbearia, construído com as mais recentes tecnologias web para proporcionar uma experiência excepcional aos clientes.

## Tecnologias Utilizadas

- **Next.js 16** - Framework React de produção
- **React 19** - Biblioteca JavaScript para interfaces de usuário
- **TypeScript** - Tipagem estática para JavaScript
- **Tailwind CSS 4** - Framework CSS utilitário
- **Framer Motion** - Biblioteca para animações

## Funcionalidades

- Design responsivo para todos os dispositivos
- Animações suaves e interativas
- Interface moderna e intuitiva
- Otimizado para performance

## Começando

Primeiro, instale as dependências:

```bash
npm install
```

Depois, execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## Scripts Disponíveis

- `npm run dev` - Executa o servidor de desenvolvimento
- `npm run build` - Cria a versão de produção
- `npm run start` - Inicia o servidor de produção
- `npm run lint` - Executa o linter
